Boilerplate

HTML5 Boilerplate provides web developers with a starting point, an HTML5, JavaScript and CSS3 foundation for building modern, cross-browser, mobile and desktop websites. By using HTML5 Boilerplate, developers can avoid reinventing the wheel, and can take advantage of the new features provided by HTML5 today without compromising usability.

The functionality provided by HTML5 Boilerplate includes cross-browser compatibility with support for Internet Explorer 6, 7 and 8, support for Modernizr, Google analytics support, IE-specific classes and more.

HTML5 Boilerplate is available for download from the HTML5Boilerplate.com , where it can be downloaded in three versions: documented, stripped and custom. The documented version is great for learning the ins and outs of HTML5 Boilerplate, the stripped version is used for actually creating a web page, and the custom version which allows the developers to select which features are included.

Features offered in HTML5 Boilerplate include cross-browser compatibility (they deal with IE6, IE7, and IE8 in a clever way); inclusion of caching and compression rules; utility classes such as .no-js and .clearfix; .png support in IE6; Modernizr support; Google analytics support; mobile browser optimizations; IE-specific classes for maximum cross-browser control; JavaScript profiling and testing support; and CDN-hosted jQuery with a local fallback script—plus quite a bit more.
